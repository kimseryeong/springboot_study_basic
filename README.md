# spring boot 공부용 

### 🏷️ 깃 커밋 컨벤션
|태그|설명|
|------|--------------------------------|
|feat|기능 추가|
|fix|버그 수정|
|refactor|코드 리팩토링|
|chore|기타 변경사항|
|comment|필요한 주석 추가 및 변경|
|docs|문서 추가, 수정, 삭제|
|init|초기 생성|
|rename|파일 혹은 폴더명 수정하거나 옮기는 작업|
|remove|파일 삭제 작업|
|design|UI 디자인 변경|
|style|코드 포맷팅, 세미콜론 추가 등 비즈니스 로직 변경 없음|
